#############
# FORECASTS #
#############
The forecasts used in this case study are for days between January 1st and December 31st, 2019. The forecasts are from the ECMWF IFS 47r1.
The description of the forecasts from this version of the ECMWF IFS can be found here:
"https://www.ecmwf.int/en/forecasts/about-our-forecasts/evolution-ifs/cycles/summary-cycle-47r1".

################
# OBSERVATIONS #
################
The observation are from NOAA and they can be downloaded from:
"https://www.ncei.noaa.gov/data/global-summary-of-the-day/archive/"
The description of the observations can be found in the following pdf README file:
"https://www.ncei.noaa.gov/data/global-summary-of-the-day/doc/readme.pdf".
